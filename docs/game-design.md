# ゲームデザイン仕様

## コアゲームループ

```
起動
  └─ 部屋の様子を確認（猫たちが自律生活中）
       └─ 任意でお世話（給餌・ブラッシング・遊び・なでる）
            └─ 猫の個性が育つ / 機嫌・絆が変化
                 ├─ 新しい迷い猫が現れる
                 ├─ 子猫誕生イベント（繁殖・任意）
                 └─ 部屋のカスタマイズ（家具・インテリア）
```

ループの肝: **「猫を観察する」だけでも楽しく、「お世話する」とより深い関係が生まれる**

---

## 猫システム

### 個性パラメータ

各猫は生成時にランダムな個性パラメータを持つ（スペクトラム形式）:

| パラメータ | 軸（低 ↔ 高） | 行動への影響 |
|-----------|--------------|-------------|
| 社交性 | 内向 ↔ 外向 | プレイヤーや他の猫への自発的な近づき方 |
| 活動性 | のんびり ↔ 活発 | 動き回る頻度、おもちゃへの食いつき |
| 愛情表現 | ツンデレ ↔ 甘えん坊 | なでられたがる度、ゴロゴロ頻度 |
| 好奇心 | 慎重 ↔ 大胆 | 新しい家具・猫への反応速度 |

### 記憶システム

- プレイヤーとのインタラクション履歴を Drift DB に蓄積
- **頻繁にお世話された猫**: 能動的にすり寄ってくる確率 UP、特定のアクション（ブラッシングなど）を自分からせがむ
- **長期間ほうっておかれた猫**: 独立心が増す（すり寄り頻度 DOWN だが機嫌は安定）
- **猫同士の関係**: 同居期間・相性パラメータから「仲良しペア」「苦手な相手」が自然発生

### 猫のステータス（Drift スキーマ）

| フィールド | 型 | 説明 |
|-----------|----|------|
| `id` | UUID | ユニーク識別子 |
| `name` | String | プレイヤーが命名 |
| `coat_color` | String | 毛色コード（繁殖で遺伝） |
| `eye_color` | String | 目の色（繁殖で遺伝） |
| `sociability` | double (0–1) | 社交性パラメータ |
| `energy` | double (0–1) | 活動性パラメータ |
| `affection` | double (0–1) | 愛情表現パラメータ |
| `curiosity` | double (0–1) | 好奇心パラメータ |
| `mood` | double (0–1) | 現在の機嫌（変動あり） |
| `health` | double (0–1) | 健康状態 |
| `bond` | int | プレイヤーとの累積インタラクション数 |
| `birth_date` | DateTime | 誕生日 |
| `parent_ids` | List\<UUID\> | 親猫の ID（繁殖産まれの場合） |

### 猫の入手方法

1. **初期の猫（1〜2 匹）** — チュートリアル終了時にプレイヤーが選択・命名
2. **迷い猫** — 定期的に窓の外に現れる。エサを置いておくと居着く（Neko Atsume 的なパッシブ収集）
3. **繁殖（任意）** — 仲の良いペアが条件を満たすと子猫誕生イベントが発生。プレイヤーが承認

## 部屋システム

### インテリア配置

- 家具をドラッグ＆ドロップで自由配置（グリッドなし・フリー配置）
- 家具カテゴリ: 寝床・爪とぎ・玩具・食器・窓際スポット・本棚・クッション・照明 等

**家具と猫行動の関係:**

| 家具 | 猫への影響 |
|------|-----------|
| 窓際マット | 日向ぼっこスポット — 時間帯で猫が集まる |
| キャットタワー | 活発な猫（energy 高）が優先使用 |
| 爪とぎポスト | ソファ等の家具へのダメージ演出を防ぐ |
| 隠れ家ボックス | 内向な猫（sociability 低）が好んで使用 |
| 暖炉 / こたつ | 冬季に全猫が集まる（特にのんびりな猫） |

### 環境変化

**時間帯（リアルタイム連動 or ゲーム内時計）:**

| 時間帯 | 光の演出 | 猫の行動傾向 |
|--------|---------|-------------|
| 朝 | 柔らかい朝日、東窓から差し込む光 | 活発に動き回る |
| 昼 | 明るい自然光 | 日向ぼっこ |
| 夕方 | 橙色の斜光 | 甘える・すり寄る |
| 夜 | 間接照明・薄暗い | 寝る・ゴロゴロ |

**季節（ゲーム内カレンダー）:**

| 季節 | 環境演出 |
|------|---------|
| 春 | 桜の花びらが窓から舞う |
| 夏 | 入道雲・風鈴 SE |
| 秋 | 落ち葉・ひんやりした光 |
| 冬 | 雪景色・こたつ登場 |

## ケアアクション

すべて **任意** — しなくてもゲームは進む。やると機嫌・絆が上昇する。

| アクション | 効果 | Rive アニメーション |
|-----------|------|-------------------|
| 給餌 | 健康維持・機嫌 UP | 食べる（ムシャムシャ）アニメ |
| ブラッシング | 絆 UP・毛並み演出 | ブラシに身を任せるアニメ |
| おもちゃ遊び | 機嫌 UP（活発な猫は大喜び） | じゃれる・跳びかかるアニメ |
| なでる（タップ） | 絆 UP（甘えん坊は目を細める） | 目を細める・ゴロゴロアニメ |
| ベッド整え | 猫が寝床を好む確率 UP | 猫が移動・くるんと丸まるアニメ |

## 繁殖システム（任意・上級者向け）

- **条件**: 同じ空間での長期同居 + 相性良好（sociability の差が小さいペア）
- **フロー**: 繁殖フラグ立つ → プレイヤーに通知（任意のため強制しない） → 承認で子猫誕生
- **遺伝**: 毛色・目色は親から確率継承（ランダム変異あり）
- **個性継承**: 個性パラメータの平均値 ± ランダム幅で子猫の個性を決定
- **子猫成長**: 誕生後 7 日（ゲーム内）で成猫へ移行

## 進行・コレクション

### 猫図鑑（Catbook）

- 出会った猫の記録（名前・誕生日・毛色・性格タイプ）
- プレイヤーが撮影した「思い出写真」をアルバムとして保存
- 繁殖で生まれた猫の家系図（任意閲覧）

### 部屋の成長

- 序盤は家具が少ない質素な部屋からスタート
- プレイ時間 / インタラクション数に応じて家具アンロック
- 「誰かの部屋感」→「自分だけの部屋感」への変化をゲーム体験として演出

## UI / UX 仕様

### 基本方針

- ミニマル UI（猫と部屋の視覚を最大化、HUD 要素は最小限）
- 必要な情報はタップ / スワイプで呼び出す（常時表示しない）
- プッシュ通知はオプトイン（デフォルト OFF）
- タイマーなし・エネルギーなし・広告なし

### 画面構成（go_router）

```
/                     ホーム（部屋ビュー・メイン）
/cat/:id              猫のプロフィール・詳細
/room/edit            部屋カスタマイズ（家具配置）
/catbook              猫図鑑・アルバム
/shop                 コンテンツショップ（コスメのみ）
/settings             設定・オプション
```

### Rive アニメーション設計

各猫の Rive ファイルにステートマシンを定義:

| ステート | トリガー |
|---------|---------|
| `idle` | デフォルト（ぼーっとする・毛づくろいなど） |
| `walk` | 移動時 |
| `sleep` | 夜・寝床使用時 |
| `play` | おもちゃ遊びアクション |
| `eat` | 給餌アクション |
| `groom` | ブラッシングアクション |
| `happy` | なでる・機嫌 MAX 時 |

## 技術仕様メモ

| 要素 | 技術 | 備考 |
|------|------|------|
| 状態管理 | Riverpod + riverpod_annotation | コード生成でボイラープレート削減 |
| ローカル DB | Drift | 猫データ・インタラクション履歴・部屋レイアウト |
| 画面遷移 | go_router | 宣言的ルーティング・ディープリンク対応 |
| アニメーション | Rive | 猫の動き・インタラクション演出 |
| ターゲット OS | Android（Flutter） | iOS 展開は将来検討 |
| ミニマム SDK | Android 21 (Lollipop) | 対象端末を広くカバー |
